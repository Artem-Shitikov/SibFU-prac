# CV Image Editor

Десктоп-приложение на Python (PySide6 + OpenCV) для обработки изображений.

## Функционал

Общий:
- Загрузка изображения из файла (png/jpg) с обработкой ошибок
- Снимок с веб-камеры с обработкой ошибок (камера недоступна/нет разрешения)
- Отображение изображения в окне
- Показ отдельного канала (R/G/B) по выбору пользователя

По варианту:
- Обрезка изображения по координатам, заданным пользователем
- Усреднение (блюр) с размером ядра, заданным пользователем
- Рисование красного круга по координатам и радиусу, заданным пользователем

## Требования

- Python 3.10+
- macOS/Linux/Windows

## Установка и запуск из исходного кода

1. Склонировать репозиторий:
   ```bash
   git clone https://github.com/Artem-Shitikov/SibFU-prac.git
   cd SibFU-prac
   ```

2. Создать и активировать виртуальное окружение:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate        # macOS/Linux
   .venv\Scripts\activate           # Windows
   ```

3. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустить приложение:
   ```bash
   python main.py
   ```

## Структура проекта

- `main.py` — точка входа, запуск приложения
- `main_window.py` — главное окно, элементы интерфейса и их обработчики
- `image_ops.py` — функции обработки изображений на OpenCV (без зависимостей от GUI)
- `requirements.txt` — список зависимостей окружения
